# Vellum

a fast, native-feeling reading app.

built fully in rust with tauri and cached hard for speed, so it stays snappy while you search, browse, and read. it's meant to compile for windows, macos, ios, and android, but i've only actually tested it on linux (arch). a web build also runs on github pages, though the real app is noticeably faster than it.

## features

- **library** keep track of what you're reading, picks up where you left off
- **discover** browse trending and filter by tags/genres
- **updates** see new chapters for series you follow
- **reader** clean, distraction-free reading
